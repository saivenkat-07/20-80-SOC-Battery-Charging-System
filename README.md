
# Smart Battery Charging System for Maintaining 20–80% SOC Using MATLAB/Simulink

## 📖 Overview

This project presents a MATLAB/Simulink-based battery charging control system that maintains the battery State of Charge (SOC) between 20% and 80% to improve battery lifespan and prevent overcharging and deep discharge.

##  Objectives

- Maintain battery SOC between 20% and 80%.
- Improve battery lifespan through controlled charging.
- Prevent overcharging and deep discharge.
- Simulate the charging strategy using MATLAB/Simulink.

##  Features

- SOC-based charging control
- MATLAB/Simulink implementation
- Automatic charging enable/disable logic
- Battery charging and discharging simulation
- Ideal component-based system model

##  Software Used

- MATLAB R2024b
- Simulink
- Stateflow

##  Repository Contents

- `SOCMAIN1.slx` – Simulink model
- `project_details.pdf` – Project presentation
